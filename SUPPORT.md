# Support

## Куда писать

- Для багов, улучшений и продуктового фидбека используйте `Issues` этого репозитория.
- Для чувствительных вопросов безопасности используйте приватный канал, описанный в [SECURITY.md](/home/minkin/minkinhub/mnplyeoman/yeoman-feedback/SECURITY.md).
- Для правок самих документов и шаблонов открывайте `Pull Request`.

## Как ускорить ответ

- используйте подходящий шаблон;
- приводите конкретные шаги и примеры;
- добавляйте контекст использования;
- не смешивайте несколько тем в одном обращении.

## Чего ожидать

Команда постарается:

- быстро подтвердить, что обращение увидели;
- приоритизировать фидбек по влиянию и срочности;
- объединять дубликаты, если тема уже обсуждается.

Сроки ответа и внедрения не гарантируются, но каждое содержательное обращение помогает улучшать Monopoly - yeoman.
